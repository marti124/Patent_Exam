# The Open-Book Examination — Companion Files

These files go with the chapter **"Reason the Constitutional Argument Yourself: The Open-Book Examination"** in the book *The Forgotten Half of the Patent Clause: How Federal Regulations Doubled the Cost of American Living and Rewrote the Constitution* by Roleigh Martin.

They are **free** for anyone reading that chapter.

## What this is for

The chapter invites you to test its constitutional argument yourself. You hand the argument's underlying materials — the Supreme Court holdings, the constitutional text, the federal regulations, and a pair of plain diagrams — to an artificial-intelligence (AI) assistant, and you ask it to reason through the question on its own. You are not asked to take anyone's word for the conclusion. You watch the AI work it out.

You do not need any special skill. Using one of these tools is like sending a text message with a file attached.

## The files

**The exam — PDF.** This is the one to start with. It works with almost any AI assistant that lets you attach a file. If you are not sure which to pick, pick this one.

**The exam — Word document.** Exactly the same exam, as an editable Word file. Choose this only if your AI assistant handles Word files more easily than PDFs.

**The scoring page — PDF.** A single page where you can write down how each AI answers, so you can compare different assistants side by side. Optional, but handy if you try more than one.

## How to run the exam

1. Download one of the exam files above (the PDF is the easy choice).
2. Open an AI assistant and start a **new** chat.
3. Tap the paper-clip or attach icon and choose the file you just downloaded.
4. Paste in one of the prompts from the chapter.
5. Read what the AI says — and, if you like, jot it on the scoring page.

Run each prompt in its own fresh chat, so each answer stands on its own. The steps are the same on a phone or a computer. The chapter walks you through everything in detail, including a beginner's guide if you have never used an AI before.
